Friday brought a keyboard swap and a surprisingly involved game of *Where in the World is This Computer?*

It started when a Practice Manager asked if we could identify the primary user of one of the devices we'd flagged for compliance. First stop was NinjaOne, where the last logged in user turned out to be our sys admin, which wasn't exactly a helpful lead. We asked Ryan if IT might already have the machine in their possession. That could've been 50/50. Next I checked the New and Retired Computers spreadsheet and found a computer with the same name listed as retired. But just when I thought I had it, the serial numbers didn't match between the spreadsheets and what NinjaOne was showing.

We went back through the tickets, found the last one filed about that specific machine, and it pointed to a manager in another department. Checked their office. Computer found!

On a smoother note, the phones for the FHA replacement next week are already provisioned and ready to go! And I also got to provision a new laptop using the same process from way back on Day 4, which at this point feels like second nature.
