## **Materials / Pre-Setup**

1. [Download Tailscale](https://tailscale.com/download) and make an account on user’s system
2. Create new Ubuntu Server VM on the Proxmox server _(Refer to [Create a VM](https://wiki.raspberrypiclub.org/books/how-to-proxmox/page/30-create-a-vm) in How to: Proxmox)_
---

## **VM Customization Options**

> [!info] Some "conventions" are only applicable to the Proxmox server at the Raspberry Pi Club - disregard "General Settings" if you're in a home lab environment unless specifically highlighted as an important feature

- General Settings _(Check advanced)_
    - Click Start at boot
    - Node: pve
    - Resource Pool: Personal 
    - VM ID: # _(any number unless following a certain convention)_
    - Name: [USER]-tailnet _(e.g. pat-tailnet)_
    - Tags: tailnet, [USER]
