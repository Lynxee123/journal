## **Materials / Pre-Setup:**

- USB Flash drive
- Machine with [minimum requirements](https://wiki.raspberrypiclub.org/books/how-to-proxmox/page/what-is-proxmox)
    - _To make a Proxmox Cluster - 2 minimum, but 3 machines are preferred (for fault tolerance and high availability)_
- Understand the schema of your network _(i.e. IP subnet, default gateway, etc.)_
---
## **Installing Proxmox**

1. Download [Proxmox Virtual Environment](https://www.proxmox.com/en/products/proxmox-virtual-environment/overview) onto any system that you're working with
2. Create a bootable USB download using a USB Creation Tool (multiple approaches stated below) 
    - [Rufus](https://rufus.ie/en/) (Windows)
    - Ventoy (Windows, macOS, Linux)
    - UNetbootin (Windows, macOS, Linux)

## **Installer Configurations**

Once you get to the Proxmox Installer, you'll be prompted with multiple things - examples/suggestions are given below:

- Choose "Install Proxmox VE (Terminal UI)"
- Hostname (FQDN): [PVE NAME].local _(use different names if you're trying to cluster - doesn't matter what you name it)_
- IP Address (CIDR): [IP ADDRESS CIDR] _(e.g. 10.0.0.100/16)_
- Gateway Address: [DEFAULT GATEWAY] _(e.g. 10.0.0.1)_
- DNS Server Address: [DNS SERVER] _(i.e. 10.0.0.1 or public DNS servers: Cloudflare - 8.8.8.8, Google - 1.1.1.1)_
